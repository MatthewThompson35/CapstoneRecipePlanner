Context - User is viewing the landing page with list of ingredients

Primary Flow
1 - user clicks on button to view ingredients
2 - system directs user to list of ingredients owned
3 - user selects an ingredient from list
4 - user clicks on button to remove ingredient
5 - user is prompted to a popup displaying that remove was successful.
6 - system updates list of ingredients


Alternative Flow(s)

NO INGREDIENT SELECTED

a4 - a popup displays to the user that an ingredient has not been selected.

return to step 2
